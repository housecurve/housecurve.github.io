## Tuning Bundle

The Tuning Bundle is an in-app purchase that provides the following features:

* **Equalize Tool** - Simple room correction.  Generate PEQ or FIR filters.  Export to Roon, Volumio, miniDSP, etc.

* **Real Time Measurement** - Pink noise analyzer.  Live tune graphic equalizers and subwoofers.  Capture moving mic measurements.

* **Advanced Plots** - Average or compare multiple sweep measurements on one plot.  Display phase and group delay.  Export impulse responses.

* **Curve Editor** - Create curves with drag and drop or edit text file directly

The Tuning Bundle can be purchased for one year (non-renewable subscription) or a lifetime unlock.
