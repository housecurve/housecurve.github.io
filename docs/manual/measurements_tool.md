# Measurements Tool
The Measurements tool is an organizer for measurements saved by the Sweep and Real Time tool.

![Measurements tool](/assets/img/measurements_tool.png "Measurements tool")
{: .app-portrait-right }


## Selection
Tap "Select" to choose measurements.

## Delete
Deletes selected measurements.

## Export
Export selected measurements as [impulse responses](file_formats.md#measurements).

Note - this feature is only available with a [Tuning Bundle](../DOWNLOAD.md#tuning-bundle) purchase.
